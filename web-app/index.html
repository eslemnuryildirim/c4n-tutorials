Helm ile Basit Web Uygulaması Deploy Etme

Bu proje stajımda Kubernetes ve Helm öğrenmek için yaptığım küçük bir çalışmadır.

Proje Hakkında

Bitnami'nin Nginx chart'ını kullanarak tek komutla bir web sunucusu ayağa kaldırdım.

Adımlar

  1. Bitnami reposunu ekledim: 
    helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update
  2. Uygulamayı kurdum: 
    helm install my-web bitnami/nginx --namespace tutorials
  3. Durumu kontrol ettim: 
    helm status my-web --namespace tutorials

Detaylar için repo'daki helm-basics.md dosyasına bakabilirsin.

Bu sayfayı görüyorsan deploy başarılı! :)
